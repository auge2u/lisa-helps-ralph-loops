## [0.1.0] - 2026-01-27

### Added
- Initial fork from ralph-it-up v1.2.0
- Renamed project to lisa-helps-ralph-loops
- Renamed plugin to lisa-loops-memory
- Memory architecture design (semantic, episodic, procedural)
- Memory file structure in `.agent/memory/`

### Changed
- Refocused project on memory persistence for multi-agent workflows
- Updated all documentation for new project identity

### Inherited from ralph-it-up v1.2.0
- Native Claude Code orchestration mode
- Quality gate validation (6 gates)
- Bash and Python validators
- Scratchpad protocol for cross-iteration context
- ralph-orchestrator v2.2.0 compatibility
